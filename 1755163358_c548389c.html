```html Hiển thị Mã HTML

Nội dung HTML của bạn đã được chuyển đổi

Dưới đây là chuỗi HTML mà bạn đã cung cấp, được hiển thị một cách trực quan và dễ đọc trong khối mã.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Generated Output</title>
  <style>
    body {
      font-family: Arial, sans-serif;
      margin: 2rem;
      background: #f5f5f5;
      color: #333;
    }
    .container {
      background: #fff;
      padding: 1.5rem;
      border-radius: 8px;
      box-shadow: 0 2px 6px rgba(0,0,0,0.1);
    }
    h1 {
      color: #007acc;
    }
    pre {
      background: #eee;
      padding: 1rem;
      border-radius: 5px;
      overflow-x: auto;
    }
  </style>
</head>
<body>
  <div class="container">
    <h1>Output</h1>
    <p>This is your generated HTML output.</p>
    <pre id="output">Your content here...</pre>
  </div>
</body>
</html>

Lưu ý: Chuỗi HTML trên đã được chuyển đổi các ký tự đặc biệt (ví dụ: < thành &lt;) để nó hiển thị dưới dạng văn bản trong trình duyệt thay vì được diễn giải thành các phần tử HTML.

```
